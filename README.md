## Pulse - Daily Summary Bot

Pulse is a Python automation project that:

- Fetches live weather information
- Retrieves a motivational quote
- Generates a daily summary report
- Saves the report as a text file
- Runs automatically using GitHub Actions

## Technologies Used

- Python
- Requests Library
- GitHub Actions
- YAML Workflows

## Automation

The bot is scheduled to run automatically every day and generates a downloadable summary artifact.
